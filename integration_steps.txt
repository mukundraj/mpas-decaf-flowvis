Steps to integrate MPAS-O with a stand alone particle tracer using Decaf
========================================================================

Path to example branch: https://github.com/mukundraj/mpas-decaf-flowvis/tree/parallel_pathlines

1. Create mpas_adapter file (mpas_adapter2.cpp): a CPP file with wrapper functions that can be used to call decaf routines from mpas fortran code.
Relative path to example file: decaf/examples/mpas_vis/src/adapter/mpas_adapter2.cpp

2. Build mpas adapter file to generate corresponding shared object file.

3. Edit MPAS-Model makefile to include the mpas adapter, decaf library, and boost shared objects.
Relative path to example file: MPAS-Model/src/Makefile

4. Edit mpas_subdriver.F
Relative path to example file: MPAS-Model/src/driver/mpas_subdriver.F

5. Edit mpas_io.F
Relative path to example file: MPAS-Model/src/framework/mpas_io.F

6. Build MPAS source by following instructions in README.md

7. Build the flow visualization code to obtain executable (flow_main) by following instructions in README.md

8. Set Decaf workflow parameters using python script (generate_script.py). Run python script.

9. Run the shell script (mpas_decaf_flowvis.sh) generated by the python script (generate_script.py).
